# Quick Start for Electron Build

## One-Command Setup

```bash
node scripts/convert-to-electron.js
```

## Manual Steps (if needed)

### 1. Install Dependencies
```bash
npm install
```

### 2. Build Next.js App
```bash
npm run build
```

### 3. Run Development
```bash
npm run electron:dev
```

### 4. Build macOS App
```bash
npm run electron:build:mac:universal
```

See ELECTRON_BUILD.md for detailed instructions.
